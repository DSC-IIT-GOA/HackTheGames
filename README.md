# HackTheGames Resources

We have created a software for automating **'Hill Climb Racing'** game 
using OpenCV module. 

In this software we'll be detecting hands and using them would be controlling the game.

## Tech Stack

- Python language 
- OpenCV module 
- Time module 
- PyautoGUI
